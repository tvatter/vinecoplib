Detailed Description

An abstract class for elliptical copula families.

This class is used in the implementation underlying the Bicop class. Users should not use AbstractBicop or derived classes directly, but always work with the Bicop interface.

Joe, Harry. Dependence modeling with copulas. CRC Press, 2014.
