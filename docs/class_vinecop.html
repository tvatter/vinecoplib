Automated model and structure selection for vine copulas

Implements the structure selection algorithm of Dissmann et al. (2013).

Parameters
datanxd matrix of copula data.
family_setthe set of copula families to consider (if empty, then all families are included; all families are included by default).
methodindicates the estimation method: either maximum likelihood estimation (method = "mle", default) or inversion of Kendall's tau (method = "itau"). When method = "itau" is used with families having more thanone parameter, the main dependence parameter is found by inverting the Kendall's tau and the remainders by profile likelihood optimization.
selection_criterionthe selection criterion; either "aic" or "bic" (default).
show_tracewhether to show a trace of the building progress (default is false).
Returns
The fitted vine copula model.
