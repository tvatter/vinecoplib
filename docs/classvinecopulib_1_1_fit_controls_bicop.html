creates the controls for fitting bivariate copula models.

Parameters
family_setthe set of copula families to consider (if empty, then all families are included).
parametric_methodthe fit method for parametric families; possible choices: "mle", "itau".
nonparametric_methodthe fit method for the local-likelihood nonparametric family (TLLs); possible choices: "constant", "linear", "quadratic".
nonparametric_multa factor with which the smoothing parameters are multiplied.
selection_criterionthe selection criterion ("aic" or "bic").
preselect_familieswhether to exclude families before fitting based on symmetry properties of the data.
