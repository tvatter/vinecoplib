Detailed Description

A class for bivariate copulas

The Bicop class is abstract, you cannot create an instance of this class, but only of the derived classes.

Select a bivariate copula

Parameters
datathe data to fit the bivariate copula.
selection_criterionthe selection criterion ("aic" or "bic").
family_setthe set of copula families to consider (if empty, then all families are included).
use_rotationswhether rotations in the familyset are included.
preselect_familieswhether to exclude families before fitting based on symmetry properties of the data.
methodindicates the estimation method: either maximum likelihood estimation (method = "mle") or inversion of Kendall's tau (method = "itau"). When method = "itau" is used with families having more than one parameter, the main dependence parameter is found by inverting the Kendall's tau and the remainders by a profile likelihood optimization.
Returns
A pointer to an object that inherits from Bicop.
